เงินกู้ (Loan)

   เงินกู้ หมายถึง เงินที่ผู้กู้ทำสัญญายืมผู้ให้กู้ โดยที่มีเงื่อนไขระยะเวลารวมถึงอัตราดอกเบื้ยที่จะต้องจ่ายคืนผู้ให้กู้รวมอยู่ในสัญญาเมื่อถึงเวลาจ่ายเงินคืนผู้ที่ให้กู้ โดยที่เงินกู้จะแบ่งเป็นสองประเภทใหญ่ๆ ก็คือ เงินกู้นอกระบบและเงินกู้ในระบบ โดยที่เงินกู้ในระบบจะเป็นการกู้จากสถาบันทางการเงินหรือธนาคาร โดยที่เป็นการกู้เงินที่อยู่ภายใต้กฎหมาย มีการให้กู้อย่างเป็นระบบมาตรฐาน ส่วนเงินกู้นอกระบบจะเป็นการการทำสัญญากู้กับบุคคลกันเอง โดยที่อาจมีสัญญาเงินกู้หรือไม่มีก็ได้ และผู้ให้กู้ยังสามารถกำหนดอัตราดอกเบี้ยเท่าใดก็ได้

การคำนวณเงินกู้รายปี

   ใช้ในการคำนวณเงินกู้ในกรณีที่อัตราดอกเบี้ยเงินกู้เป็นรายปีและต้องจ่ายเงินกู้ทุกเดือน

ดอกเบี้ยเงินกู้ต่อเดือน = จำนวนเงินที่กู้ * ((อัตราร้อยละดอกเบี้ยรายปี / 100 ) / 12)

จำนวนเงินที่ต้องจ่ายต่อเดือน = ดอกเบี้ยเงินกู้ต่อเดือน + ( จำนวนเงินที่กู้ / (12 * จำนวนปีที่กู้) )
